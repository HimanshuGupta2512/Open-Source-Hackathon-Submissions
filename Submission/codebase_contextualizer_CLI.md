## Project Details

**Project Title:** Codebase Contextualizer CLI

**Project Description:** Codebase Contextualizer CLI is a high-performance, completely local semantic search engine for your codebase. It solves the critical privacy issue of sending proprietary, sensitive code to third-party cloud APIs. Instead of relying on basic regex string matching, this tool uses Abstract Syntax Tree (AST) parsing to intelligently chunk code by functions, classes, and methods. It leverages on-device machine learning to generate vector embeddings directly on the local CPU, storing them in an optimized local database (SQLite WAL) for sub-millisecond similarity search. This empowers developers to search their code using natural language with zero latency and zero data leaks.

**Tech Stack Used:** Node.js, Commander.js, Worker Threads, web-tree-sitter (WASM), better-sqlite3, @xenova/transformers (all-MiniLM-L6-v2)
